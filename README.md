## Installation

Install my-project with npm

```bash
  git clone https://github.com/wahyunandanovan/todolist-imp.git
  cd todolist-imp
  npm Install
  npm run dev
```
    
## Tech Stack

**Client:** Nextjs, Chakra UI, Axios,React Query, React Hooks Form
